# Greek Grammar Exercise Templates / Πρότυπα Ασκήσεων

Use these templates to create drills for learners. Adjust difficulty to CEFR level if the user provides it.

## Article and Case Drill

```markdown
## Άσκηση: Άρθρα και πτώσεις
Συμπλήρωσε το σωστό άρθρο.

1. Βλέπω ___ φίλο μου.
2. Μιλάω με ___ δασκάλα.
3. Το βιβλίο ___ παιδιού είναι εδώ.
4. Πάω σε ___ σχολείο.
5. ___ Μαρία διαβάζει.

## Απαντήσεις
1. τον — direct object, masculine accusative singular.
2. τη/την — after **με**, feminine accusative singular.
3. του — possession, neuter genitive singular.
4. το → στο if contracted: **στο σχολείο**.
5. Η — subject, feminine nominative singular.
```

## Verb Aspect Drill

```markdown
## Άσκηση: Όψη ρήματος
Διάλεξε τη σωστή μορφή.

1. Θέλω να (γράφω / γράψω) ένα email τώρα.
2. Θέλω να (διαβάζω / διαβάσω) κάθε βράδυ.
3. Μην (μιλάς / μιλήσεις) όταν εξηγεί ο δάσκαλος.
4. Θα (βλέπω / δω) την ταινία αύριο.
5. Όταν ήμουν μικρός, (πήγα / πήγαινα) συχνά στο χωριό.

## Απαντήσεις
1. γράψω — bounded/completed action.
2. διαβάζω — habitual action.
3. μιλάς — ongoing/prohibitive: stop talking / don’t talk during class.
4. δω — single future event.
5. πήγαινα — repeated past habit.
```

## Clitic Pronoun Drill

```markdown
## Άσκηση: Αντωνυμίες
Ξαναγράψε την πρόταση με αδύνατη αντωνυμία.

1. Βλέπω τον Γιώργο.
2. Έδωσα το βιβλίο στη Μαρία.
3. Θα πω την αλήθεια σε εσένα.
4. Μην κάνεις αυτό.
5. Δώσε το βιβλίο σε εμένα.

## Απαντήσεις
1. Τον βλέπω.
2. Της το έδωσα.
3. Θα σου την πω.
4. Μην το κάνεις.
5. Δώσ’ το μου. / Δώσε μου το βιβλίο.
```

## Error Correction Quiz

```markdown
## Διόρθωσε τα λάθη
Βρες και διόρθωσε το λάθος σε κάθε πρόταση.

1. Με ο φίλος μου πήγαμε σινεμά.
2. Που είσαι;
3. Θέλω πάω σπίτι.
4. Η καλός μέρα αρχίζει νωρίς.
5. Μου αρέσει οι ταινίες.

## Απαντήσεις
1. Με **τον φίλο μου** πήγαμε σινεμά. — accusative after **με**.
2. **Πού** είσαι; — question word takes tonos.
3. Θέλω **να πάω** σπίτι. — Greek uses **να + verb**.
4. Η **καλή** μέρα αρχίζει νωρίς. — adjective agreement.
5. Μου **αρέσουν** οι ταινίες. — verb agrees with **οι ταινίες**.
```

## CEFR Scaling

- **A1:** articles, present tense, basic word order, common verbs, everyday vocabulary.
- **A2:** cases after prepositions, weak pronouns, past/future basics, common aspect contrasts.
- **B1:** aspect in subordinate clauses, clitic combinations, relative clauses, formal vs informal variants.
- **B2:** nuanced aspect, register, complex noun phrases, participles, genitive alternatives.
- **C1/C2:** style, idiom, precision, advanced syntax, formal written Greek, ambiguity management.
