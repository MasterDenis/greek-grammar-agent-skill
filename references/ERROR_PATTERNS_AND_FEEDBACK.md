# Error Patterns and Feedback Strategies / Συχνά Λάθη και Στρατηγικές Διόρθωσης

Use this file when giving learner feedback or diagnosing recurring mistakes.

## Feedback Principles

1. Correct the sentence first.
2. Explain the highest-impact errors, not every possible stylistic improvement.
3. Separate grammar from naturalness.
4. Give one reusable rule per error when possible.
5. End with a corrected example the user can imitate.

## Error Category Labels

Use concise labels:

- **Tonos/spelling**
- **Article/case agreement**
- **Noun gender**
- **Adjective agreement**
- **Verb aspect**
- **Verb person/number**
- **Particle choice**
- **Negation**
- **Pronoun/clitic placement**
- **Word order/focus**
- **Register/naturalness**

## Common Learner Error Patterns

### Missing or Wrong Tonos

Error:

- **Που εισαι;**

Better:

- **Πού είσαι;**

Rule:

- Most multisyllabic words need tonos. Question **πού** and **πώς** take tonos.

### Article Omitted Before Names or Abstract Nouns

Error:

- **Μαρία είναι εδώ.**

Better:

- **Η Μαρία είναι εδώ.**

Rule:

- Greek commonly uses the definite article before names and general nouns.

### Wrong Case After Preposition

Error:

- **με ο φίλος μου**

Better:

- **με τον φίλο μου**

Rule:

- Most common prepositions take accusative.

### Wrong Gender

Error:

- **ο πόλη**

Better:

- **η πόλη**

Rule:

- Noun gender must be learned with the noun; article and adjective follow it.

### Adjective Agreement Error

Error:

- **η καλός μέρα**

Better:

- **η καλή μέρα**

Rule:

- Adjectives agree with the noun in gender, number, and case.

### English Infinitive Transfer

Error:

- **Θέλω πάω σπίτι.**

Better:

- **Θέλω να πάω σπίτι.**

Rule:

- Modern Greek uses **να + verb** where English often uses an infinitive.

### Wrong Negation

Error:

- **Θέλω δεν πάω.**

Better:

- **Δεν θέλω να πάω.** or **Θέλω να μην πάω.**

Rule:

- **δεν** negates indicative verbs. **μην** negates subjunctive and command-like structures.

### Aspect Error with `να`

Error:

- **Θέλω να γράφω αυτό το γράμμα.** when the meaning is “I want to write/finish this letter.”

Better:

- **Θέλω να γράψω αυτό το γράμμα.**

Rule:

- Use perfective aspect for a bounded, completed action.

### Wrong Clitic Placement

Error:

- **Βλέπω τον.** meaning “I see him.”

Better:

- **Τον βλέπω.**

Rule:

- Weak object pronouns usually come before finite verbs.

### Overusing Subject Pronouns

Acceptable but less natural:

- **Εγώ πάω στο σπίτι και εγώ τρώω.**

Natural:

- **Πάω στο σπίτι και τρώω.**

Rule:

- Greek drops subject pronouns when they are clear from the verb ending, unless emphasis is needed.

### `μου αρέσει` Agreement

Error:

- **Μου αρέσει οι ταινίες.**

Better:

- **Μου αρέσουν οι ταινίες.**

Rule:

- The liked thing is the grammatical subject, so the verb agrees with it.

## Correction Formats

### Minimal Correction

Use when the user asks “is this correct?”

```markdown
Better: **...**
Reason: **...** takes accusative after the preposition **...**.
```

### Detailed Learner Feedback

```markdown
**Corrected:** ...

**Main corrections:**
1. **Article/case:** ...
2. **Verb aspect:** ...
3. **Naturalness:** ...

**Pattern to remember:** ...
```

### Native-Speaker Style Polish

```markdown
**Natural version:** ...

**More formal:** ...
**More casual:** ...

The original is grammatically understandable, but ...
```

## Exercise Feedback

When grading exercises:

1. Mark each answer as **Correct**, **Almost**, or **Incorrect**.
2. Give the corrected form.
3. Explain the rule in one sentence.
4. Track repeated errors and summarize at the end.

Example:

```markdown
1. **Almost:** με ο φίλος μου → **με τον φίλο μου**
   Rule: **με** takes accusative, so **ο φίλος** becomes **τον φίλο**.

Repeated pattern: accusative after prepositions.
```

## Register Labels

Use these labels when alternatives differ by context:

- **Neutral standard:** safe in most contexts.
- **Colloquial:** natural in speech/messages.
- **Formal:** suitable for essays, official writing, news, academic style.
- **Literary/archaic:** rare in ordinary Modern Greek.
- **Dialectal/regional:** not general standard Greek.

## Do Not Overstate

Avoid saying:

- “This is always wrong” when it is merely formal, colloquial, or regionally marked.
- “Greek word order is free” without explaining focus and naturalness.
- “Passive voice” for every `-μαι` verb.
- “Aorist equals past” without explaining perfective aspect.
