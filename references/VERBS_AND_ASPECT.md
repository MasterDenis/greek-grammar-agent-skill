# Verbs and Aspect / Ρήματα και Όψη

## The Central Idea: Aspect

Modern Greek verb grammar is built around **aspect** as much as tense.

- **Imperfective aspect**: action as ongoing, repeated, habitual, or unbounded.
- **Perfective aspect**: action as complete, single, bounded, or viewed as a whole.
- **Perfect aspect**: result or current relevance of a completed action, often with **έχω**.

Examples:

- **γράφω** = I write / I am writing.
- **έγραφα** = I was writing / I used to write.
- **έγραψα** = I wrote, viewed as a completed event.
- **θα γράφω** = I will be writing / will write repeatedly.
- **θα γράψω** = I will write once/complete it.
- **έχω γράψει** = I have written.

## Main Tense-Aspect Forms

| Form | Typical meaning | Example |
|---|---|---|
| Present | present ongoing/habitual | γράφω |
| Imperfect / Paratatikos | past ongoing/habitual | έγραφα |
| Aorist | past completed/bounded | έγραψα |
| Future imperfective | future ongoing/repeated | θα γράφω |
| Future perfective | future completed/single | θα γράψω |
| Perfect | completed with current relevance | έχω γράψει |
| Pluperfect | completed before past reference point | είχα γράψει |
| Future perfect | will have completed | θα έχω γράψει |

## Past Forms and Augment

The past often adds **έ-** when the verb form would otherwise have fewer than three syllables:

- γράφω → **έγραφα**, **έγραψα**
- λύνω → **έλυνα**, **έλυσα**

In plural forms, the augment may disappear in common speech/writing if stress rules allow:

- **γράψαμε** / **εγράψαμε**; the first is normal in everyday Modern Greek.
- **πήγαμε**, not usually **επήγαμε** except formal/literary.

## Perfective Stem

Many verbs have a present/imperfective stem and a perfective/aorist stem:

| Present | Aorist | `να` perfective | Meaning |
|---|---|---|---|
| γράφω | έγραψα | να γράψω | write |
| διαβάζω | διάβασα | να διαβάσω | read/study |
| ανοίγω | άνοιξα | να ανοίξω | open |
| βλέπω | είδα | να δω | see |
| λέω | είπα | να πω | say |
| τρώω | έφαγα | να φάω | eat |
| πίνω | ήπια | να πιω | drink |
| παίρνω | πήρα | να πάρω | take |
| πηγαίνω/πάω | πήγα | να πάω | go |
| έρχομαι | ήρθα | να έρθω | come |
| βρίσκω | βρήκα | να βρω | find |

## Subjunctive with `να`

The particle **να** combines with dependent verb forms. It can use imperfective or perfective aspect:

- Θέλω **να γράφω** κάθε μέρα. = I want to write every day / habitually.
- Θέλω **να γράψω** ένα γράμμα. = I want to write a letter, complete it.

Common triggers:

- θέλω να...
- μπορώ να...
- πρέπει να...
- χρειάζεται να...
- πριν/αφού/όταν depending on meaning

## Future with `θα`

- **θα γράφω** = I will be writing / I will write repeatedly.
- **θα γράψω** = I will write / finish writing.
- **θα έχω γράψει** = I will have written.

`θα` also appears in conditional-like forms:

- **θα ήθελα** = I would like.
- **θα μπορούσα** = I could / would be able to.

## Negation

Use **δεν** for indicative forms:

- **δεν γράφω**
- **δεν έγραψα**
- **δεν θα γράψω**

Use **μην/μη** with subjunctive, imperative-like prohibitions, and non-indicative complements:

- **μην γράψεις** = don’t write.
- Θέλω **να μην πας**.
- **μη μιλάς** = don’t talk / stop talking.

## Imperative

Modern Greek imperatives distinguish aspect:

### Imperfective Imperative

Used for ongoing/repeated instructions:

- **γράφε** = keep writing / write regularly.
- **μη μιλάς** = don’t talk / stop talking.

### Perfective Imperative

Used for one completed action:

- **γράψε** = write it.
- **διάβασε** = read it.
- **άνοιξε** = open it.

Plural/polite:

- **γράψτε**, **διαβάστε**, **ανοίξτε**.

For prohibitions, Greek often uses **μην + subjunctive**, not a direct negative imperative:

- **Μην το κάνεις.** = Don’t do it.
- **Μην πας.** = Don’t go.

## Voice: Active and Non-Active

Modern Greek traditionally uses active and passive labels, but the so-called passive forms can express passive, reflexive, reciprocal, or middle meanings.

Examples:

- **γράφω** = I write.
- **γράφεται** = it is written / it is being written.
- **πλένομαι** = I wash myself.
- **συναντιόμαστε** = we meet each other.
- **κοιμάμαι** = I sleep; non-active form, not passive meaning.

Do not translate every `-μαι` form as passive.

## Present Non-Active Endings

Example **γράφομαι**:

| Person | Form |
|---|---|
| εγώ | γράφομαι |
| εσύ | γράφεσαι |
| αυτός/ή/ό | γράφεται |
| εμείς | γραφόμαστε |
| εσείς | γράφεστε / γραφόσαστε |
| αυτοί/ές/ά | γράφονται |

Common deponent/non-active verbs:

- **έρχομαι** = I come.
- **κάθομαι** = I sit.
- **κοιμάμαι** = I sleep.
- **θυμάμαι** = I remember.
- **φοβάμαι** = I fear.

## Participles

Common participle types:

- Active present adverbial: **γράφοντας**, **μιλώντας** = while/by writing, speaking.
- Passive perfect participle/adjective: **γραμμένος**, **κλεισμένος**, **κουρασμένος**.

Examples:

- **Περπατούσε μιλώντας στο τηλέφωνο.**
- **Η πόρτα είναι κλεισμένη.**

## Aspect Choice Examples

| Greek | Meaning |
|---|---|
| Θέλω να διαβάζω κάθε βράδυ. | I want to read every night. |
| Θέλω να διαβάσω αυτό το βιβλίο. | I want to read/finish this book. |
| Όταν πήγαινα σχολείο... | When I used to go to school... |
| Όταν πήγα στο σχολείο... | When I went to the school... |
| Μη μιλάς. | Don’t talk / stop talking. |
| Μη μιλήσεις. | Don’t say anything / don’t speak once. |

## Correction Strategy

When correcting verb errors:

1. Identify intended time: present, past, future, conditional-like, completed result.
2. Identify aspect: ongoing/repeated vs single/completed.
3. Select the right stem: imperfective or perfective.
4. Check person and number.
5. Check particle compatibility: `δεν`, `μην`, `να`, `θα`, `ας`.
6. Check clitic placement around the verb.
7. Preserve register: **μιλώ** vs **μιλάω**, **γράφουν** vs **γράφουνε**.
